# 00-Inbox

这里是所有新信息的统一入口。不要在这里长期囤资料，每天或每周至少清理一次。

## 放什么

- 临时摘录
- 网页链接
- 还没判断用途的想法
- 等待整理的课程、文章、截图说明

## 清理规则

处理每条内容时问 3 个问题：

1. 它属于哪个项目、领域或资源？
2. 一周内会不会用到？
3. 不保存会不会真的损失？

如果答案不清楚，优先删除或只保留一句自己的转述。

## 去向

- 有明确目标和截止日期：移动到项目目录。
- 属于长期能力建设：移动到对应学习领域。
- 暂时用不上但未来可能有价值：移动到资源目录或主题目录。
- 已过期或完成：移动到 `04-Archives`。
