# 后端转前端快速上手路线图

> 目标：先能独立做页面，再能接接口，最后做一个完整小项目。

## 1. HTML + CSS

- **时间**：5 - 7 天
- **学到什么程度**：能独立写一个静态页面，布局不乱，手机和电脑都能看
- **重点**
  - 语义标签
  - 盒模型
  - Flex
  - Grid
  - 响应式布局
- **资料**
  - [MDN HTML 入门](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content)
  - [MDN CSS 基础](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Styling_basics)
  - [web.dev Learn HTML](https://web.dev/learn/html)
  - [web.dev Learn CSS](https://web.dev/learn/css)

## 2. JavaScript

- **时间**：7 - 10 天
- **学到什么程度**：能让页面动起来，能做点击、表单、列表渲染、接口请求
- **重点**
  - 变量
  - 函数
  - 数组和对象
  - DOM
  - 事件
  - Promise
  - `async/await`
- **资料**
  - [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
  - [web.dev Learn JavaScript](https://web.dev/learn/javascript)

## 3. TypeScript

- **时间**：3 - 5 天
- **学到什么程度**：能看懂前端项目里的类型，能给函数、接口、组件加类型
- **重点**
  - `type`
  - `interface`
  - 泛型
  - 类型收窄
  - 基础项目配置
- **资料**
  - [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)

## 4. React

- **时间**：10 - 14 天
- **学到什么程度**：能自己做一个带登录、列表、详情、增删改查的页面
- **重点**
  - 组件
  - props
  - state
  - Hooks
  - 条件渲染
  - 列表渲染
  - 表单
  - 请求
  - 路由
- **资料**
  - [React Learn](https://react.dev/learn)

## 5. 项目实战

- **时间**：7 - 10 天
- **学到什么程度**：能独立做一个小后台，接自己的后端接口
- **推荐项目**
  - Todo
  - 博客后台
  - 简单 CRM
  - 管理台

## 最短执行顺序

1. HTML + CSS
2. JavaScript
3. TypeScript
4. React
5. 项目实战

## 每阶段及格线

- HTML + CSS：能画页面
- JavaScript：能让页面动
- TypeScript：能写得稳
- React：能做业务
- 项目实战：能独立串起来

## 适合每天怎么学

- **1 - 2 小时/天**
- 先看一点资料
- 马上自己写
- 每个阶段至少做一个小练习
