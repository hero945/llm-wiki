# 前端学习路线图 - 14 天版

> 目标：配合 AI 写页面，先做到能看懂、能修改、能调样式。

## 学习原则

- 先会看懂代码
- 再会改样式
- 再会接接口
- 最后会和 AI 配合写页面

## 第 1 - 3 天：HTML + CSS

### 第 1 天：HTML 基础

- [ ] 学标签、标题、段落、链接、图片、列表
- [ ] 练习写一个个人简介页
- [ ] 目标：看懂页面结构

### 第 2 天：表单 + 结构

- [ ] 学 form、input、button、语义标签
- [ ] 练习写一个登录页静态结构
- [ ] 目标：知道页面大概怎么分区

### 第 3 天：CSS 基础

- [ ] 学选择器、颜色、字体、边距、盒模型
- [ ] 把前两天页面美化一下
- [ ] 目标：会改样式

## 第 4 - 5 天：布局

### 第 4 天：Flex

- [ ] 学横向排列、居中、分布
- [ ] 做一个顶部导航栏
- [ ] 目标：会摆页面

### 第 5 天：Grid + 响应式

- [ ] 学 Grid 基本用法、手机适配
- [ ] 做一个卡片列表页
- [ ] 目标：页面在手机和电脑都能看

## 第 6 - 8 天：JavaScript

### 第 6 天：JS 基础

- [ ] 学变量、函数、数组、对象
- [ ] 写几个小函数
- [ ] 目标：能看懂基础 JS

### 第 7 天：DOM + 事件

- [ ] 学 querySelector、点击事件、表单提交
- [ ] 做一个点击计数器
- [ ] 目标：知道页面怎么动起来

### 第 8 天：异步

- [ ] 学 Promise、`async/await`、fetch
- [ ] 请求一个接口并把数据渲染到页面
- [ ] 目标：能看懂接口请求

## 第 9 - 10 天：React

### 第 9 天：React 组件

- [ ] 学组件、props、state
- [ ] 把一个页面拆成 2 - 3 个组件
- [ ] 目标：能看懂 React 页面

### 第 10 天：Hooks

- [ ] 学 `useState`、`useEffect`
- [ ] 做一个列表页，支持加载数据
- [ ] 目标：能自己改 React 页面

## 第 11 - 12 天：TypeScript

### 第 11 天：TS 基础

- [ ] 学类型、接口、类型别名
- [ ] 给前面的函数和对象加类型
- [ ] 目标：看懂项目里的类型

### 第 12 天：TS + React

- [ ] 学组件类型、props 类型
- [ ] 给 React 页面补类型
- [ ] 目标：能在项目里用 TS

## 第 13 - 14 天：AI 协作实战

### 第 13 天：让 AI 帮你写页面

- [ ] 练习把需求说清楚
- [ ] 让 AI 生成页面
- [ ] 自己修改样式和布局
- [ ] 目标：会和 AI 配合

### 第 14 天：做一个小项目

- [ ] 做登录页 + 列表页 + 详情页
- [ ] 尝试接自己的后端接口
- [ ] 目标：能跑通一个完整页面流程

## 14 天后的状态

- [ ] 能看懂基础前端代码
- [ ] 能改页面样式和布局
- [ ] 能接简单接口
- [ ] 能跟 AI 配合完成一个页面
