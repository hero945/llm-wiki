## 什么是运算符？

本章节主要说明Python的运算符。举个简单的例子 **4 + 5 = 9** 。 例子中， **4** 和 **5** 被称为 **操作数 (Operands)** ，+ 称为运算符 (Operator)。

简单来说，运算符就是用于对一个或多个值（操作数）执行特定运算，从而得出结果的符号。

Python支持多种类型的运算符，每种类型都负责不同的任务。下面我们将逐一学习这些主要的运算符类别。

## 🔢 一元算术运算符 (Arithmetic Operators)

这类运算符用于执行标准的数学运算。这是初学者最先接触到的运算符组。

### 1. 加法 (+) 和减法 (-)
| 符号 | 名称 | 描述 | 示例代码 |
| :---: | :---: | :--- | :--- |
| `+` | 加号 | 计算两个操作数的和。 | `5 + 3` $\rightarrow$ `8` |
| `-` | 减号 | 计算第一个操作数与第二个操作数的差。 | `10 - 4` $\rightarrow$ `6` |

### 2. 乘法 (*) 和除法 (/)
| 符号 | 名称 | 描述 | 示例代码 |
| :---: | :---: | :--- | :--- |
| `*` | 乘号 | 计算两个操作数的积。 | `4 * 5` $\rightarrow$ `20` |
| `/` | 除法 | 执行浮点数除法（结果总是浮点数）。 | `10 / 3` $\rightarrow$ `3.333...` |

### 3. 取余 (%)、幂运算 (**) 和整除 (//)

| 符号 | 名称 | 描述 | 示例代码 |
| :---: | :---: | :--- | :--- |
| `%` | 取余号 | 计算两个操作数相除后的**余数**。 | `10 % 3` $\rightarrow$ `1` (因为 $10 = 3 \times 3 + 1$) |
| `**` | 幂运算符 | 表示指数运算（第一个操作数取第二个操作数的次方）。 | `2 ** 3` $\rightarrow$ `8` ($2^3$) |
| `//` | 整除运算符 | 执行整数除法，结果会向下取整。 | `10 // 3` $\rightarrow$ `3.0` (注意：如果操作数包含浮点数，结果仍为浮点数) |

#### 📝 示例代码：
```python
a = 17
b = 5

print(f"{a} + {b} = {a + b}")  # 加法
print(f"{a} % {b} = {a % b}")  # 取余 (余数)
print(f"2 ** 4 = {2 ** 4}")    # 幂运算 (16)
print(f"{a} // {b} = {a // b}") # 整除 (3.0)
```

## ⚖️ 二元比较运算符 (Comparison Operators)

这类运算符用于**比较**两个操作数是否相等、大小关系等。它们执行的结果永远是布尔值 `True` 或 `False`。

| 符号 | 名称 | 描述 | 示例代码 | 结果 |
| :---: | :---: | :--- | :--- | :---: |
| `==` | 等于号 | 判断两个操作数是否**相等**（值是否相同）。 | `5 == 5` | `True` |
| `!=` | 不等于号 | 判断两个操作数是否**不相等**。 | `4 != 6` | `True` |
| `>` | 大于号 | 判断左侧是否大于右侧。 | `7 > 3` | `True` |
| `<` | 小于号 | 判断左侧是否小于右侧。 | `2 < 1` | `False` |
| `>=` | 大于等于号 | 判断左侧是否大于或等于右侧。 | `5 >= 5` | `True` |
| `<=` | 小于等于号 | 判断左侧是否小于或等于右侧。 | `3 <= 2.9` | `False` |

#### 📝 示例代码：
```python
x = 10
y = 10

print(f"x == y: {x == y}") # True (值相等)
print(f"x != y: {x != y}") # False (值不相等)
print(f"x > 5: {x > 5}")   # True
```

## 💡 逻辑运算符 (Logical Operators)

这类运算符用于组合多个布尔表达式（即包含 `True` 或 `False` 的表达式），用于构建复杂的判断条件。

| 符号 | 名称 | 描述 | 用途场景 |
| :---: | :---: | :--- | :--- |
| `and` | 与 | 当**所有**操作数都为 `True` 时，结果才为 `True`。 | 判断“必须满足A **并且** B”。 |
| `or` | 或 | 只要**任一**操作数为 `True`，结果就为 `True`。 | 判断“满足 A **或者** B 就行”。 |
| `not` | 非 | 对布尔值进行取反（将 `True` 变为 `False`，反之亦然）。 | 反转一个条件判断的结果。 |

#### 📝 示例代码：
```python
# 假设用户年龄大于18岁 (is_adult) 和是否持有证件 (has_id)
is_adult = True
has_id = False

# and: 年龄必须大，并且要有证件才能购票
can_buy = is_adult and has_id  # True and False -> False
print(f"可以购买吗? {can_buy}") # 输出：False

# or: 满足年龄或有证件就可以通过
pass_test = (10 > 5) or (2 == 3) # True or False -> True
print(f"是否通过测试？ {pass_test}") # 输出：True

# not: 反转条件
result = not can_buy  # 不是可以购买的状态
print(f"状态反转后是: {result}") # 输出：True
```

## 🚀 其他重要运算符 (The Rest)

除了上述核心的算术、比较和逻辑运算外，Python还提供了几个非常重要的用于判断对象关系的特殊运算符。

### 1. 赋值运算符 (Assignment Operators)
这类运算符通常用于将右侧的值赋给左侧的变量，但它们也提供了一种**简写**的方式来执行更复杂的计算赋值。

| 符号 | 等价于 | 说明 | 示例代码 | 效果 |
| :---: | :---: | :--- | :---: | :--- |
| `=` | N/A | 最基本的赋值操作。 | `x = 10` | 将值赋给变量 `x`。 |
| `+=` | `x = x + y` | 加并赋值：将右侧与左侧相加，结果重新赋值给左侧。 | `x += 5` | 等于 `x = x + 5`。 |
| `-=` | `x = x - y` | 减并赋值。 | `y -= 2` | 等于 `y = y - 2`。 |

#### 📝 示例代码：
```python
count = 10
print(f"原始 count: {count}") # 10

# 使用 += 进行高效的累加
count += 5
print(f"使用 += 后的 count: {count}") # 15 (相当于 count = count + 5)
```

### 2. 身份运算符 (`is` 和 `is not`)
用于判断两个变量是否**指向内存中的同一个对象**（即它们是否是同一块内存地址）。这与 `==` 的比较值不同。

*   `a is b`: 判断 a 和 b 是否为同一个对象。
*   `a is not b`: 判断 a 和 b 是否不是同一个对象。

#### 📝 示例代码：
```python
# 情况一：两个变量指向完全相同的值和内存地址
list1 = [1, 2]
list2 = list1 # 让 list2 指向 list1 的内存地址

print(f"list1 == list2: {list1 == list2}")  # True (值相等)
print(f"list1 is list2: {list1 is list2}")  # True (指向同一对象！)

# 情况二：创建两个独立的对象
a = [1, 2]
b = [1, 2] # 虽然内容一样，但 Python 通常会分配不同的内存地址

print(f"a == b: {a == b}")                # True (值相等)
print(f"a is b: {a is b}")                # False (通常指向不同对象)
```

### 3. 成员运算符 (`in` 和 `not in`)
用于判断一个元素是否属于某个序列（如列表、元组、字符串等）。

*   `x in sequence`: 判断 x 是否在序列中。
*   `x not in sequence`: 判断 x 是否不在序列中。

#### 📝 示例代码：
```python
fruits = ["apple", "banana", "cherry"]

# 检查是否包含元素
print(f"'banana' in fruits: {'banana' in fruits}") # True

# 检查是否不包含元素
print(f"'grape' not in fruits: {'grape' not in fruits}") # True
```

## Python位运算符

按位运算符是把数字看作二进制来进行计算的。Python中的按位运算法则如下：

下表中变量 a 为 60，b 为 13，二进制格式如下：

```
a = 0011 1100

b = 0000 1101

-----------------

a&b = 0000 1100

a|b = 0011 1101

a^b = 0011 0001

~a  = 1100 0011
```

| 运算符 | 描述 | 实例 |
| --- | --- | --- |
| & | 按位与运算符：参与运算的两个值,如果两个相应位都为1,则该位的结果为1,否则为0 | (a & b) 输出结果 12 ，二进制解释： 0000 1100 |
| \| | 按位或运算符：只要对应的二个二进位有一个为1时，结果位就为1。 | (a \| b) 输出结果 61 ，二进制解释： 0011 1101 |
| ^ | 按位异或运算符：当两对应的二进位相异时，结果为1 | (a ^ b) 输出结果 49 ，二进制解释： 0011 0001 |
| ~ | 按位取反运算符：将二进制表示中的每一位取反，0 变为 1，1 变为 0。~x 类似于 -x-1 | (~a ) 输出结果 -61 ，二进制解释： 1100 0011 (以补码形式表示)，在一个有符号二进制数的补码形式。 |
| << | 左移动运算符：运算数的各二进位全部左移若干位，由 << 右边的数字指定了移动的位数，高位丢弃，低位补0。 | a << 2 输出结果 240 ，二进制解释： 1111 0000 |
| \>> | 右移动运算符：把">>"左边的运算数的各二进位全部右移若干位，>> 右边的数字指定了移动的位数 | a >> 2 输出结果 15 ，二进制解释： 0000 1111 |

以下实例演示了Python所有位运算符的操作：

## 实例(Python 2.0+)

a = 60 b = 13 c = 0 c = a & b; print "1 - c 的值为：", c c = a | b; print "2 - c 的值为：", c c = a ^ b; print "3 - c 的值为：", c c = ~a; print "4 - c 的值为：", c c = a << 2; print "5 - c 的值为：", c c = a >> 2; print "6 - c 的值为：", c

以上实例输出结果：

```

## Python身份运算符

身份运算符用于比较两个对象的存储单元
